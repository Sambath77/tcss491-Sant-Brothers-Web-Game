Sant Brothers

Instructions

1. Controller

Key D / Right Arrow Key ->: Move Forward
Key A / Left Arrow Key <-: Move Backward
Key X: Jump
Key C: Shoot

2. Objective

The game has two levels which have different obstacles.

The player can upgrade the power guns when touching Angel and Panda. Angel will give the player a random weapon and Panda will give the player an extra life and weapon.
